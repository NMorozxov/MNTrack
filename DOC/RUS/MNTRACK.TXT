 2. Подключение нодлистов.
 ─────────────────────────

     Формат конфигурционного файла компилятора нодистов  mntrnc.ctl :
     Каждая строка содержит ключевое слово или начинается с символа ';' 
     что означает строку коментарий.
     Ключевые слова :
      NODELIST <n>
       Нодлист
       Где <n> имя нодлиста
      NODELISTMASK <n>
       Где <n> маска (Имя нодиста с символами ? и * ) по которой будет 
       производиться поиск нодлистов.
      NODELISTMASKNUMERIC <n>
       Тоже самое что и NODELISTMASK только будут компилироваться только те
       нодлисты которые имеют цифровые расширения.
      NODELISTSEGMENT <z> <s>
       Сегмент нодлиста 
        <z> Зона или начальный адрес 
        <s> Сегмент
      ADDRESS <a>
       Адрес <a> тоже будет считаться listed

     ─ ВHИМАHИЕ! Hе включайте пойнтлисты в mntrnc.ctl !!! ─

     Компиляция  осуществляется  запуском  mntrnc.exe.  В  pезультате будет
     создан  индекc-файл  - mntrack.ndx. Возможны следующие ключи командной
     строки:
      /H - Выводит список ключей.

     Утилитой  NodeAdd  можно  добавить  адpес в mntrack.ndx, чтобы он тоже
     считался listed.


 3. Сканиpование нетмейла.
 ─────────────────────────

     Для  сканирования  нетмейла  запyскается  файл  mntrack.exe,  возможны
     следующие ключи командной строки :
      /H - Выводит список ключей.
      /Q - Во время сканирования не выводить ничего на экран.
      /F <mask> - сканировать файлы <mask> вместо тех что указаны в 
                  конфигурвционном файле
      /S <script> - использовать всего один скрипт <script>
      /U <fromdir> <todir> - Распаковать PKT файлы из католога <fromdir>
                  в каталог <todir>. ВНИМАНИЕ при распаковке PKT файлов
                  MNTrack не транзитные письма атрибут TRS, рекомендуется
                  сделать это первым скриптом.

     Коды возврата (errorlevels):
      0 - Работа MNTrack завершена успешно.
      1 - Ошибка при загрузке mntrack.cfg.
      2 - Нехватка памяти.
      3 - Неизвестный ключ командной строки.
      4 - Ошибка при загрузке mntrack.ndx.
      5 - Невозможно открыть скрипт.
      6 - Ошибка при загрузке mntrack.dmn.
      7 - Ошибка открытия или создания log-файла.
      8 - Ошибка при загрузке mntrack.aka.
      9 - Ошибка при сохранении конфигурации.
