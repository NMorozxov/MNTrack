MNTic
FIDO Net Файлэхо процессор для Win 95/98/NT.

Возможности:

  • MNTic использует Win32 API
  • MNTic ипсользует системные функции копирования файлов
  • GUI Сетап
  • MNTIC.DMN полностью совместим MNTRACK.DMN (от MNTrack)



checkmail
Программа антивирусной проверки с помощью Kaspersky Antivirus входящей почты помещаемой в UNIX майлбоксы программой mail.local.

  • Совместима с любым MTA
  • Примеры инсталяции для sendmail/postfix
  • Легко дописать для другого AV
